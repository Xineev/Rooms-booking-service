# RoomsBooking

API для бронирования переговорных комнат.

## Стек

- ASP.NET Core 9
- Entity Framework Core
- PostgreSQL
- Docker
- Swagger

## Возможности

- CRUD для комнат
- CRUD для бронирований
- Глобальная обработка ошибок
- Result<T> для обработки бизнес-ошибок

## В разработке

- JWT Authentication
- Authorization
